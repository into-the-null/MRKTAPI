# MRKTAPI - Telegram Market API Library


## Features

- 🔐 Authentication and session management
- 💰 Balance checking
- 🎁 Gift searching and purchasing
- 📊 Price floors and market data
- 🏆 Giveaway management
- 🛒 Selling and inventory management
- 🚀 Async/await support
- 🛡️ Comprehensive error handling

pip install mrktapi
